//: INFO: Info Zeile  
// Erste Zeile der Datei. Bei allen anderen Daten ist die Reihenfolge frei.  

   Spalte              Format            Beschreibung  
 /*  1 -  4 */ char[ 4]; // Satzkennung ‘INFO’  
 /*  7 - 31 */ char[25]; // Kennung erzeugendes Programm/Version  
 /* 32 - 69 */ char[38]; // Datum/Uhrzeit letzte Änderung durch Programm  
 /* 70 - 73 */ int     ; // [4] Kennung spezieller Berechnungsmodus: '0' oder fehlend(' ') : Standard / konventionell 1 : Berechnung Spanntabellen für Ausland/Export ? Braucht man noch?   

  



//: INFS: Info-Zeile slc (nur in slc-Datei enthalten)   

// *** Neu ab Version 3.7.4 ***

/*  7 - 22 */ char[16+1];   // Datum/Uhrzeit letzte Änderung durch Programm (Format tag.monat.jahr stunde:minute)  
/* 24 - 39 */ char[16+1];   // Datum/Uhrzeit 263-Eingabedatei (Format tag.monat.jahr stunde:minute)

/*
Anmerkung: Falls in allgemeinen Einstellungen Windgesetz nach DIN EN 50341 eingestellt ist 
(siehe oben, Datenzeile PAR1, Spalte 34) 
werden die Staudruckdaten anders interpretiert. 
Anstatt des wirklichen Staudruckes werden in den entsprechenden Datenfeldern die Bezugsstaudrücke gelesen, 
aus denen erst der reale Staudruck im Berechnungskern ermittelt wird. 
Steht hier eine '-1', so wird der Bezugsstaudruck aus der Formel '100+7*H/6' gemäß DIN EN 50341 ermittelt. 
Des Weiteren werden die Höhenstufen dann als Schwelle der Geländehöhe für den Mastfußpunkt interpretiert. 
D.h. ab diesem Wert für den Mastfußpunkt gilt der zugehörige Wert für den Bezugsstaudruck.  
*/
